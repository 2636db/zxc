오늘의 목표: 행동 대신 말로 표현하기

지도 원리: 학생이 손으로 끌거나 가리키기 전에, 2~5초 기다리고 그림 버튼을 보여 준 뒤 단어 말하기를 모델링합니다. 말, 비슷한 발음, 음절 일부, AAC 버튼 선택도 처음에는 모두 의사소통 시도로 인정하고 즉시 강화합니다.

사용 순서
① 보여 주기 → ② 듣기 → ③ 따라 말하기 → ④ 바로 들어주기

  1. 기다리기 2~5초
  2. 그림 가리키기
  3. 단어 첫소리 말해주기
  4. 모델링 “물”
  5. 말하면 즉시 제공

2022 교육과정 기반 단계별 말하기 프로그램

핵심 목표는 요구·거부·도움·감정을 기능적인 단어로 표현하여, 잡아끌기·가리키기·울기·소리지르기를 대체하는 것입니다. 학생의 현재 수준을 고려하여 공통 국어 성취기준은 개별화하고, 특수교육 기본 교육과정의 의사소통 참여 목표와 연결했습니다.

1단계. 요구 단어 말하기

목표: 원하는 물건이나 행동을 단어로 말한다.

  • 목표어: 물, 과자, 쉬, 더, 주세요, 열어
  • 기회: 물 마시기 전, 간식 전, 놀잇감 잠시 멈춤
  • 절차: 기다리기 → 그림 제시 → “물” 모델링 → 반응 즉시 제공
  • 숙달: 2일 연속 80% 이상 독립 또는 최소촉진

2단계. 선택 단어 말하기

목표: 두 가지 중 원하는 것을 단어로 고른다.

  • 목표어: 공, 자동차, 블록, 책, 음악
  • 기회: “공? 자동차?” 실제 물건 2개 제시
  • 오류교정: 잘못 누르면 다시 보여 주고 정답 모델링
  • 확장: 단어가 안정되면 “공 주세요”로 확장

3단계. 거부·중단 단어 말하기

목표: 울거나 소리 지르기 전에 싫음과 중단을 말한다.

  • 목표어: 싫어, 그만, 아니, 쉬어요
  • 기회: 덜 선호 과제, 전환, 기다리기 상황
  • 중요: 거부 표현은 무시하지 말고 짧은 선택권 제공
  • 예: “싫어” → “쉬어요? 도와줘?” 선택

4단계. 도움·감정 단어 말하기

목표: 어려움과 감정을 단어로 알린다.

  • 목표어: 도와줘, 아파, 좋아, 무서워, 화나요
  • 기회: 뚜껑 열기, 퍼즐 맞추기, 대기 상황
  • 강화: 도움 요청 시 즉시 부분 도움 제공
  • 일반화: 보건실, 급식실, 통합학급에서도 사용

12주 운영안

기간지도 초점목표 단어자료·촉진평가
1~2주기초 반응·선호도 평가물, 쉬, 더실물+그림, 즉각 강화반응 형태 기록
3~4주요구 표현주세요, 열어, 과자시간지연, 첫소리 촉진독립률 50% 목표
5~6주선택 표현공, 차, 블록, 책2선택, 오류교정선택 정확도
7~8주거부·중단 표현싫어, 그만, 아니대체반응훈련, 기능적 의사소통문제행동 전 표현 여부
9~10주도움·감정 표현도와줘, 아파, 좋아, 화나요상황카드, 역할놀이상황별 사용
11~12주일반화·유지개별 핵심어 8~12개교실·급식·화장실·가정촉진 감소, 유지율

시간상황단어반응메모

가정·통합학급 연계 안내

학교에서 배운 단어를 같은 그림, 같은 말, 같은 반응으로 반복해야 일반화가 빠릅니다. 보호자와 통합학급 교사에게 아래 문장을 그대로 공유해도 됩니다.

가정에서

  • 원하는 것을 바로 주기 전에 2초 기다립니다.
  • “말로 해요. 물.”처럼 짧게 모델링합니다.
  • 비슷하게 말해도 바로 들어줍니다.
  • 울 때는 길게 설명하지 말고 그림을 보여줍니다.

통합학급에서

  • 책상 위에 핵심 단어 4개만 둡니다.
  • 친구가 대신 말하지 않도록 잠깐 기다립니다.
  • 말하면 “잘 말했어”라고 짧게 칭찬합니다.
  • 과제 전환 전 “끝, 쉬어요, 도와줘”를 확인합니다.

문제행동 대응

  • 울기·소리지르기는 의사표현 실패 신호로 봅니다.
  • 안전 확보 후, 가장 쉬운 대체 단어를 제시합니다.
  • 단어 표현이 나오면 즉시 기능을 충족합니다.
  • 진정 후 같은 상황을 아주 짧게 다시 연습합니다.
